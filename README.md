### 1. 프로젝트 소개

  약속 구독 어플

    우리는 현재 언택트 시대로 인해 소통의 부재를 겪고 있다. 
    때문에 코로나 블루등 부정적인 요인들이 증가하고 있어서 
    사람들간의 소통을 위해 이 어플을 개발하기로 주제를 정했다.

  기능 
    
    - 로그인을 통한 개인별 캘린더 약속 관리
    - 구독리스트에 친구 추가
    - 친구 캘린더에서 시간별 약속 신청
    - 기타 추가 예정

  기술
    
    - Flutter
    - Dart
    - Firebase

  최종산출물
